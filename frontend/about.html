About PPE Sentry

Real-time PPE Compliance Detection System

System Overview

PPE Sentry is an AI-powered real-time Personal Protective Equipment (PPE) compliance monitoring system designed for industrial and construction environments. The system uses computer vision to automatically detect whether workers are wearing required safety equipment and alerts supervisors to violations in real-time.

The system processes live video feeds from CCTV cameras, applies YOLOv8 object detection to identify workers and their PPE, and generates instant alerts for non-compliance events. All detection events are logged for compliance reporting and audit trails.

AI Model & Detection Capabilities

Model: YOLOv8 (Ultralytics)

The system uses a custom-trained YOLOv8 model optimized for PPE detection in industrial settings. The model achieves sub-50ms inference time on GPU, enabling real-time processing of multiple camera streams.

Violation Classes (Configurable per User)

Users can configure which PPE items trigger violations through Admin Settings → Detection Settings. By default, all 5 PPE violations and Fall Detection are enabled. The system uses class IDs 6-10 for PPE violations and class ID 0 for Fall Detection.

Technology Stack

Backend

  • Python 3.10+ / FastAPI
  • YOLOv8 (Ultralytics)
  • OpenCV for video processing
  • Supabase (PostgreSQL)
  • JWT Authentication

Frontend

  • Vanilla JavaScript (ES6+)
  • Chart.js for analytics
  • Lucide Icons
  • CSS Grid/Flexbox layout
  • Served via FastAPI StaticFiles

AI/ML

  • YOLOv8n/s/m/l custom weights
  • PyTorch / ONNX Runtime
  • Real-time MJPEG streaming
  • Configurable violation classes
  • Sub-50ms inference (GPU)

Database & Auth

  • Supabase (PostgreSQL)
  • Row Level Security (RLS)
  • JWT tokens (HS256)
  • Bcrypt password hashing
  • REST API via PostgREST
